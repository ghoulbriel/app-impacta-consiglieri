# app-impacta-consiglieri
This repository is dedicated for the Graduation Project (TCC) at Faculdade Impacta. Consiglieri is a platform designed to collect and manage customer feedback for commercial establishments, leveraging Artificial Intelligence to generate detailed analyses and strategic reports.

# funcionalidades
- Criar review: voltado para o cadastro do feedback enviados por clientes por diferentes canais de atendimento.
